## 🗺 Navigating Your Map

The map view helps you find information quickly. Your code appears as circles connected by lines.

* Click and drag your mouse to move around the canvas.

* Use the search bar at the top to find a specific file by name.

The tool updates the view in real time. If you move your mouse over a connection, the tool highlights the relationship between two parts of your code. This helps you see how one function calls another.

## 🔍 Understanding the AI Analysis

graphify-dotnet uses artificial intelligence to read your files. It looks for patterns in how you name your functions and how you organize your projects. This allows it to make smart guesses about what parts of your code are most important. 

The tool categorizes your code into three main groups:

1. Core files: These files contain the logic that makes your software work.
2. Support files: These files help the core files run.
3. Libraries: These are external parts of your code that add extra features.

You can see these groups highlighted in different colors on the map. This makes it easy to spot where you have complex segments or extra files you no longer need.
